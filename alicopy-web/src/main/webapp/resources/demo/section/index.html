主图视频制作


请按 “右方向键（→）” 继续

登录之后到后台主页

选择商品的列表中的某个商品

你会发现

在右侧点击"生成视频"按钮

该按钮变成"正在生成.."表示已经证在后台生成,请耐心等候!

当右侧出现"下载视频"按钮,点击该按钮

上传视频文件

点击搜索条上的"上传视频"按钮

跳出的淘宝视频上传页面,上传即可

这时，点击红杏图标就能

立即解决 这些域名的访问障碍

如果实在不行

你也可以打开“一直”模式

这时，所有访问都会通过红杏

（这能解决99%的问题，不过副作用是访问普通网站会变慢）

这些就是红杏的主要功能了

是不是很简单呢？
